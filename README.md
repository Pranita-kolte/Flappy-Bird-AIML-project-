# 🐦 Flappy Bird AI using Deep Q-Network (DQN)

This project implements a Deep Q-Network (DQN) agent that learns to play Flappy Bird using Reinforcement Learning.

## Features

- Deep Q-Network (DQN)
- Experience Replay
- PyTorch implementation
- Gymnasium environment
- Pygame rendering

## Project Structure

```
agent.py                 # Training agent
dqn.py                   # Neural network
experience_replay.py     # Replay buffer
flappy_bird_game.py      # Game environment
parameters.yaml          # Hyperparameters
```

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python flappy_bird_game.py
```

## Technologies Used

- Python
- PyTorch
- Gymnasium
- Pygame
- Reinforcement Learning
